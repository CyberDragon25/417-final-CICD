=================================
May need to set up Java 11 or Java 8 if on pyrite:

>export java11=/usr/lib/jvm/java-11/bin/java
>export javac11=/usr/lib/jvm/java-11/bin/javac

Then replace java, javac below with $java11, $javac11

(can do the same for Java 8 which is /usr/lib/jvm/java-1.0.8/bin/java or javac)

===================================

>javac Tutorial_Stack/src/main/java/tutorial/*.java

>java -jar evosuite-1.2.0.jar -projectCP Tutorial_Stack/src/main/java -class tutorial.Stack

Test cases will be in the evosuite-tests directory
Reports will be in the evosuite-reports directory

export CLASSPATH=.:evosuite-tests:lib/junit-4.13.jar:lib/hamcrest-core-1.3.jar:Tutorial_Stack:evosuite-standalone-runtime-1.2.0.jar
